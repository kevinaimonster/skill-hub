# 技能宝 SkillHub

中文社区最大的 AI 技能搜索、安装与智能推荐平台。

## 安装技能宝

```bash
npx skills add kevinaimonster/skill-hub -g -y
```

安装后，你可以随时对 AI 说「帮我找一个做 XX 的技能」来搜索和安装新技能。

## 特性

- **中文优先** — 所有技能推荐默认中文展示
- **安全可信** — 每个技能通过 12 重安全检测
- **智能推荐** — 在你需要时自然推荐，不硬塞广告
- **去中心化** — 直接通过 AI 搜索安装，无需访问网页
